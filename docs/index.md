# Alter/Ego

Alter/Ego is a lightweight text-to-speech assistant designed for quick desktop automation and scripting.

## Installation

```bash
pip install alter-ego
```

## What is Included

- A Python API with a minimal `AlterEgo` class
- A Click-based CLI exposed as `alter-ego`
- Packaging, CI, and documentation scaffolding
