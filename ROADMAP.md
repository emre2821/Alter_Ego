# Roadmap

This document outlines planned features and improvements for Alter/Ego. Items are subject to change based on community feedback and contributor availability.

## Current Focus (v0.2.0)

### Developer Experience
- [ ] Add devcontainer configuration for VS Code
- [ ] Add Docker support for containerized deployment
- [ ] Improve test coverage to 80%+

### Documentation
- [ ] Add API reference documentation
- [ ] Create video tutorials for common workflows
- [ ] Add internationalization (i18n) guide

## Near-Term Goals (v0.3.0)

### Features
- [ ] Support for additional TTS engines (Azure, Google Cloud)
- [ ] Voice profile presets (speed, pitch, volume)
- [ ] Audio file export (save speech to WAV/MP3)

### Integration
- [ ] REST API for remote control
- [ ] WebSocket support for real-time streaming
- [ ] Plugin architecture for custom extensions

## Long-Term Vision (v1.0.0)

### Core Improvements
- [ ] Async/await support for non-blocking speech
- [ ] Speech queue management
- [ ] Event hooks for speech lifecycle

### Ecosystem
- [ ] PyPI package publishing automation
- [ ] Homebrew formula for macOS
- [ ] Snap/Flatpak packages for Linux

### Community
- [ ] Contributor recognition program
- [ ] Regular release cadence (monthly)
- [ ] Community Discord/Discussions

## Contributing to the Roadmap

Have ideas for features or improvements? We'd love to hear from you!

1. **Open a discussion**: Share your ideas in GitHub Discussions
2. **Submit an issue**: Create a feature request with details
3. **Contribute directly**: Pick an item and submit a PR

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.
