# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please email the maintainer at [REPLACE_WITH_EMAIL] with the following details:

- Description of the issue and potential impact
- Steps to reproduce
- Any suggested mitigations

## Response Timeline

We aim to acknowledge new reports within **5 business days**. Verification and remediation timelines depend on severity, but we will provide updates at least every two weeks until resolution.

## Supported Versions

The current released version on PyPI is supported with security updates. Older versions may receive fixes at the maintainer's discretion.
